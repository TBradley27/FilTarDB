About

The purpose of FilTarDB is to provide a database of animal microRNA target predictions with accompanying sequencing information, so that the biological relevance of target predictions can be easily ascertained. More specifically, data derived from RNA-Seq experiments is used to determine the relative abundance of mRNA transcripts for a given biological context, and also generate context-specific 3'UTR annotations accounting for the effects of alternative polyadenylation and cleavage. Data generated using the FilTar command line tool is used for FilTarDB. As described, FilTar uses the context++ (cf TargetScan7) and miRanda algorithms for microRNA target prediction.

How to use

  1. Select a species of interest from the drop-down menu
  2. Select a biological context of interest from the second box which appears on the page
  3. Select either a microRNA of interest, a gene of interest or both
  4. Selects a TPM threshold
  5. Select one or multiple microRNA target prediction algorithms
  6. Submit query

The results of the query are presented in the form of a table. Results can be ordered according to each table column, by clicking on the relevant column header. The whole table can be searched for specific key words or strings using the search field. There are widgets underneath the table allowing the user to download the entire results table in various file formats, and also a widget to allow the user to iterate through different set of records contained within the results table.
